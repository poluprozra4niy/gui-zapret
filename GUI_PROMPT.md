# Профессиональный Промт для Создания GUI Zapret (Deep Packet Inspection Bypass)

**Роль:** Senior Full-Stack Developer & UI/UX Expert.

**Задача:**
Разработать современный, "премиальный" графический интерфейс (GUI) для консольной утилиты `winws.exe` (Zapret), которая используется для обхода замедления Discord и YouTube. Текущее решение работает через `.bat` файлы, что неудобно для конечного пользователя.

**Контекст:**
Проект находится в папке с уже существующими бинарниками (`bin/winws.exe`, `bin/WinDivert.dll`) и списками доменов (`lists/`). Наша цель — заменить или дополнить `.bat` файлы удобным приложением.

---

## 1. Технический Стек
Для достижения максимального визуального качества и кроссплатформенности (Windows target):
*   **Core:** Electron (для обертки системных вызовов).
*   **Frontend:** React + TypeScript.
*   **Styling:** TailwindCSS (использование градиентов, blur-эффектов, анимаций).
*   **System Integration:** Node.js `child_process` для запуска `winws.exe` и управления службами Windows (`sc`).

## 2. Дизайн-система (Aesthetics)
*   **Тема:** Dark Mode (Глубокий синий/фиолетовый фон, акценты Neon Blue/Cyan).
*   **Стиль:** Glassmorphism (полупрозрачные панели, размытый фон).
*   **Анимации:** Плавное появление, индикаторы статуса (пульсирующая точка online/offline).
*   **Шрифты:** Modern Sans-serif (Inter или JetBrains Mono для логов).

## 3. Функциональные Требования

### A. Главный Экран (Dashboard)
1.  **Статус:** Большая кнопка/индикатор "ЗАЩИТА АКТИВНА" / "ЗАЩИТА ОТКЛЮЧЕНА".
    *   При нажатии запускает/останавливает процесс `winws.exe`.
2.  **Выбор режима (Preset Selector):**
    *   Выпадающий список стратегий (на основе логики из `general.bat`, `general (ALT).bat` и т.д.).
    *   Примеры пресетов: "Discord + YouTube (General)", "Только Discord", "Alt Strategy 1", "Fake Packet Mode".
3.  **Быстрая диагностика:**
    *   Кнопки "Проверить YouTube" и "Проверить Discord" (делают curl/ping запрос).

### B. Редактор Списков (Config Manager)
1.  Вкладка для редактирования файлов:
    *   `lists/list-general.txt`
    *   `lists/list-exclude.txt`
2.  Простой текстовый редактор с подсветкой синтаксиса или список с кнопками Add/Remove.

### C. Управление Службой (Service Mode)
1.  Тоггл "Запускать вместе с Windows".
2.  Логика: Выполняет команды `sc create` / `sc delete` или вызывает существующий `service.bat`.

### D. Логи (Console)
1.  Выдвижная консоль снизу, показывающая вывод `stdout`/`stderr` от `winws.exe` в реальном времени.

## 4. Логика работы (Implementation Details)
*   Приложение должно искать `bin/winws.exe` относительно своего расположения.
*   Перед запуском стратегии нужно завершать предыдущие процессы `winws.exe`.
*   При закрытии приложения спрашивать: "Свернуть в трей" или "Выключить обход".
